# **App Name**: Maze Canvas

## Core Features:

- Maze Rendering: Displays the dynamic labyrinth on an HTML5 Canvas using the provided JavaScript drawing logic.
- Dynamic Maze Generation Tool: Utilizes a generative AI tool to create diverse and complex maze layouts with varying difficulty or aesthetic themes.
- Generate New Maze: A user interface element that allows triggering the generation and rendering of a new maze, showcasing the AI's capabilities.

## Style Guidelines:

- Primary accent color: Vibrant cosmic blue (#5c8fff), used for interactive elements and key highlights, evoking a sense of depth and advanced technology.
- Background color: Deep, desaturated navy (#14171a) with a subtle blue tint, forming an immersive dark canvas for the maze display.
- Accent color: Bright cyan (#6bcbef), sparingly applied for complementary emphasis and visual contrast against the primary blue.
- Body and headline font: 'Space Grotesk' (sans-serif), chosen for its modern, techy, and slightly futuristic aesthetic that aligns with code-driven visuals.
- Minimalist, geometric icons for user controls such as 'generate' or 'reset', featuring clean lines that harmonize with the labyrinth's structured visual style.
- Centered and responsive layout with the HTML5 Canvas as the focal point, ensuring clear visibility and engagement across different device screen sizes.
- Subtle pulsating glow on the maze walls, integrating the dynamic effects from the provided drawing code to imbue the labyrinth with a living, ethereal quality.
